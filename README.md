# Portfolio ni Lian

A personal portfolio website built with HTML and CSS as part of a Web Development course project.

## 📁 Project Structure

    portfolio-ni-lian/
    ├── index.html
    ├── resume.html
    ├── projects.html
    ├── certifications.html
    ├── hire-me.html
    ├── css/
    │   └── style.css
    └── images/
        ├── lian-luigi-paredes.jpg
        └── lian-luigi-paredes-cv.pdf

## 📄 Pages

- **About** — Introduction, What I Do, and Technical Skills
- **Resume** — Education, Experience, and Affiliations
- **Projects** — Showcase of personal and academic projects
- **Certifications** — Earned certifications from DataCamp, Cisco, and others
- **Hire Me** — Contact form and availability status

## 🛠️ Built With

- HTML5
- CSS3 (External Stylesheet)
- Google Fonts — Inter
- Git & GitHub for version control
